Introduction

Python is a high-level, interpreted programming language that emphasizes simplicity and code readability.
Its clear and concise syntax makes it ideal for both beginners and experienced developers. 


print("Hello, World")
Variables

Variables in Python are used to store values, and their types are automatically determined at runtime.
Python supports several data types, which we'll see in a while. Python has no command for declaring variables. 


x = 5
y = "John"
print(x)
print(y)
Data types

In programming, data type is an important concept, variables can store data of different types, and different types can do different things. 


# Most used types
integer_var = 1
float_var = 1.25
string_var = "text"
bool_var = True

Python has the following data types built-in by default, in these categories:

  • Text type: str
  • Numeric types: int, float, complex
  • Sequence Types: list, tuple, range
  • Mapping Type: dict
  • Set Types: set, frozenset
  • Boolean Type: bool
  • Binary Types: bytes, bytearray, memoryview
  • None Type: NoneType
Control structures

Python offers various control structures to manage program flow, such as conditionals and loops.

Conditionals include if, elif, and else statements, which allow you to execute code based on certain conditions.

Loops, such as for and while, enable you to repeat code multiple times.

Additionally, Python provides control statements like break, continue, and pass to further manage the flow within loops. 


numero = 1
while True: # Loops
    # Conditionals
    if numero == 10:
        break # Control Satatements
    else:
        numero = numero + 1
Functions

Functions allow you to reuse code and improve the organization of programs.

They are defined using the def keyword, followed by the function name and parentheses.

Functions can take arguments and return values. Here's an example of a simple function:


def greet(name):
    return f"Hello, {name}!"

print(greet("Alice"))

In this example, the greet function takes one argument, name, and returns a greeting message. You can call this function with different arguments to get different results.
